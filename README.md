# TIGI HR Resume Auto-Filler (Recruiter Edition)

This extension is specifically built for recruiters using the TIGI HR portal. It automates the candidate addition process by extracting data from CVs using Gemini AI.

## Installation
1.  Go to `chrome://extensions/`.
2.  Enable **Developer mode**.
3.  **Load unpacked** -> Folder: `c:\Users\rohan\Downloads\resume automation extention\TIGI-HR-Resume-AutoFiller`.

## Configuration
- **API Key**: Enter your Google AI Studio (Gemini) API Key in the popup. It will be saved automatically for future use.
- **Model**: Default is `gemini-3.1-flash-lite-preview`. You can change this to any Gemini model you have access to (e.g., `gemini-1.5-flash`).

## Usage
1.  Open the **Add Resume** page on TIGI HR.
2.  Click the extension icon.
3.  Upload a CV and click **Extract & Fill Form**.
