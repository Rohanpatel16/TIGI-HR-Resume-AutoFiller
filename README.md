# TIGI HR Resume Auto-Filler (Recruiter Edition) 🚀

A premium Chrome extension designed for high-performance recruiters. This tool automates the candidate addition process on the TIGI HR portal by extracting structured data from CVs using Google's **Gemini AI**.

![Design Preview](https://github.com/Rohanpatel16/TIGI-HR-Resume-AutoFiller/raw/main/icons/icon128.png)

## ✨ Features

- **Gemini AI Parsing**: Instantly extracts Name, Email, Mobile, Experience, Skills, and more from any PDF/DOCX CV.
- **Glassmorphism UI**: A beautifully crafted, modern dark-themed interface built with a pro recruiter's workflow in mind.
- **Keyboard Shortcut (`Ctrl+Shift+E`)**: Trigger the extraction process without even reaching for your mouse.
- **Intelligent Staging**: The extension remembers your last uploaded CV even if you close the popup or switch tabs.
- **High-Precision Timer**: Track extraction performance in real-time.
- **Secure API Masking**: Elegant, non-intrusive CSS masking for your Gemini API key.
- **Drag & Drop Support**: Simply drop a CV into the window to start.

## 📥 Installation

1.  Clone this repository or download the ZIP.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (top-right toggle).
4.  Click **Load unpacked** and select the extension folder.

## ⚙️ Configuration

1.  **API Key**: Obtain a free Gemini API key from [Google AI Studio](https://aistudio.google.com/).
2.  **Model**: Default is `gemma-3-27b-it`. You can experiment with other models like `gemini-1.5-flash` or `gemini-1.5-pro`.
3.  **Relevant Experience**: Toggle this to calculate total experience vs. experience specifically matching the primary designation.

## 🖱️ Usage

1.  Navigate to the **Add Resume** page on the TIGI HR portal.
2.  Click the extension icon or press `Ctrl+Shift+E`.
3.  Drop a CV or click to browse.
4.  Click **✨ Extract & Fill Form** and watch the fields populate in seconds.
